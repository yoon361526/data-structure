# 자료구조 예제 정리

## 소개

이 저장소는 자료구조 수업에서 학습한 예제 코드를 정리한 저장소입니다.
C언어를 이용하여 기본 자료구조와 탐색 알고리즘을 구현하고, 각 프로그램의 동작 구조를 확인하는 것을 목적으로 합니다.

## 개발 환경

* Language: C
* Editor: Visual Studio Code
* Compiler: GCC
* Debugger: GDB
* OS: Windows

## 학습 내용

이 저장소에는 다음과 같은 자료구조 및 알고리즘 예제가 포함되어 있습니다.

| 분류       | 내용                               |
| -------- | -------------------------------- |
| 탐색       | 이진 탐색                            |
| 스택       | 배열 스택, 연결 리스트 스택                 |
| 큐        | 원형 큐, 연결 큐                       |
| 연결 리스트   | 단순 연결 리스트, 원형 연결 리스트, 이중 연결 리스트  |
| 다항식      | 연결 리스트를 이용한 다항식 덧셈               |
| 트리       | 배열 기반 이진 트리, 스레드 이진 트리, 이진 탐색 트리 |
| 그래프 탐색   | 깊이우선탐색, 너비우선탐색                   |
| 최소 신장 트리 | 크루스칼 알고리즘, 프림 알고리즘               |
| 최단 경로    | 다익스트라 알고리즘                       |

## 주요 파일 설명

| 파일 또는 주제      | 설명                                              |
| ------------- | ----------------------------------------------- |
| Binary Search | 정렬된 배열에서 중앙값을 기준으로 탐색 범위를 줄이며 값을 찾는 예제          |
| Stack         | LIFO 구조를 가지는 스택의 삽입과 삭제 연산 예제                   |
| Queue         | FIFO 구조를 가지는 큐의 삽입과 삭제 연산 예제                    |
| Linked List   | 노드를 포인터로 연결하여 데이터를 저장하는 연결 리스트 예제               |
| CList         | 마지막 노드가 첫 번째 노드를 가리키는 원형 연결 리스트 예제              |
| DList         | 이전 노드와 다음 노드를 모두 가리키는 이중 연결 리스트 예제              |
| Poly          | 연결 리스트를 이용하여 다항식의 항을 저장하고 덧셈을 수행하는 예제           |
| Binary Tree   | 배열 또는 포인터를 이용하여 이진 트리를 표현하고 순회하는 예제             |
| BST           | 이진 탐색 트리의 삽입, 삭제, 탐색 기능을 구현한 예제                 |
| DFS           | 그래프에서 한 경로를 끝까지 탐색한 뒤 되돌아오는 깊이우선탐색 예제           |
| BFS           | 큐를 이용하여 가까운 정점부터 탐색하는 너비우선탐색 예제                 |
| Kruskal       | 간선의 가중치를 기준으로 최소 비용 신장 트리를 구하는 예제               |
| Prim          | 하나의 시작 정점에서 최소 비용 신장 트리를 확장하는 예제                |
| Shortest Path | 다익스트라 알고리즘을 이용하여 시작 정점에서 다른 정점까지의 최단 거리를 구하는 예제 |

## 실행 방법

각 C 파일은 GCC를 이용하여 컴파일한 뒤 실행할 수 있습니다.

```bash
gcc 파일이름.c -o 파일이름.exe
./파일이름.exe
```

예시:

```bash
gcc stack.c -o stack.exe
./stack.exe
```

Visual Studio Code에서 디버깅을 사용하는 경우 `.vscode/tasks.json`과 `.vscode/launch.json` 설정을 통해 GCC/GDB 환경에서 실행할 수 있습니다.
